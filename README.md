## Chatbot using AI

This project is an AI-powered chatbot designed to provide intelligent, context-aware responses through a user-friendly web interface. The chatbot processes user queries in real time by integrating the Gemini AI API to generate accurate and relevant responses.

## Features
- Interactive chatbot interface built using HTML, CSS, and JavaScript  
- Context-aware responses powered by Gemini AI API  
- Efficient handling of high-volume user queries  
- Reliable response generation with proper error handling  

## Tools & Technologies
- Frontend: HTML, CSS, JavaScript  
- AI Integration: Gemini AI API
  
## Workflow
1. Users submit queries via the chatbot interface.  
2. Requests are sent to the Gemini AI API for processing.  
3. The chatbot returns intelligent, context-aware responses in real time.
